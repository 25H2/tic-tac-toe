\# Tic-Tac-Toe



\## Описание

Простая консольная игра "Крестики-нолики" на поле 3x3 для двух игроков.



\## Как запустить

1\. Открой main.cpp в Visual Studio.

2\. Собери проект и запусти (Ctrl+F5).
